4. VALIDATION: Utilization Calculation Audit
Problem: Ensure utilization calculation is accurate and consistent.

Requirements:

Audit and confirm the following formula is correctly implemented:
Available Hours (per day) = 
  IF (day is within contract period 
      AND day is a working day 
      AND day is not a holiday 
      AND day is not vacation/absence)
  THEN daily_hours (e.g., 8h for 40h/week employee)
  ELSE 0

Billable Utilization = Billable Hours / Available Hours
Overall Utilization = Total Hours / Available Hours
Calculation must happen per day, then aggregate for week/month/period
Edge cases to handle:
Part-time employees (e.g., 20h/week = 4h/day)
Mid-week contract start (e.g., starts Wednesday)
Mixed holidays (employee A has holiday, employee B doesn't)
Add unit tests or validation scripts for these edge cases
